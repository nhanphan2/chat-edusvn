# Chatbot API với Firestore

API chatbot thay thế Google Sheets, sử dụng Firestore Database.

## Features
- Exact match search
- Similarity matching
- Analytics logging
- CORS support

## API Endpoints
- GET/POST: `/api/chatbot?q=câu hỏi&userId=user123`

## Deploy
- Vercel: Auto-deploy từ GitHub
- Database: Firebase Firestore
